html BASIC TAG TABLE

Tag Name

Tag explain

Links to Doc
<!DOCTYPE> Defines the document type and HTML version MDN Web Docs
<html> Represents the root of an HTML or XHTML document MDN Web Docs
<body> Represents the content of an HTML document MDN Web Docs
<header> The <header> tag defines the header section on a document. MDN Web Docs
<nav> The <nav> tag contains the navigation links for the website. MDN Web Docs
<main> The <main> tag contains the main content of the web page. MDN Web Docs
<article> Use the <article> tag to define self-contained sections in a document or website. MDN Web Docs
<aside> The <aside> tag defines some content aside from the content it is placed in. MDN Web Docs
<section> The <section> tag defines a section in a document. MDN Web Docs
<figcaption> Represents a caption or a legend associated with a figure or an illustration described by the rest of the data of the <figure> element which is its immediate ancestor MDN Web Docs
<figure> The <figure> tag specifies self-contained content, like illustrations, diagrams, photos, code listings, etc. MDN Web Docs
<footer> The <footer> tag defines a footer for a document or section. MDN Web Docs
<table> The element that surrounds all other table elements. MDN Web Docs
<tr> Defines a row of cells in the table. MDN Web Docs
<th> Defines a heading cell to label a row or column. MDN Web Docs
<td> Defines a single piece of data in the table—a cell. MDN Web Docs
